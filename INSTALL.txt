FIRST TIME USAGE
===============================================================

STEP 1: Check Prerequisites
----------------------------
Your model MUST have a nesting root with these attributes:
- Group with attribute: ABF > 'is-nesting-root' = true
- Contains nested boards/sheets

To check:
1. Select your nesting root group
2. Right-click > Entity Info > Advanced Attributes
3. Look for: ABF > is-nesting-root = true

===============================================================
TROUBLESHOOTING
===============================================================

ERROR: "N1 Nesting Root not found"
---------------------------------------------------------------
Solution: Your model needs a nesting root group
1. Ensure you have a group with ABF > is-nesting-root = true
2. Check Entity Info > Advanced Attributes


ERROR: Plugin doesn't load
---------------------------------------------------------------
Solution: Check Ruby Console for error messages
1. Open Extensions > Ruby Console
2. Look for red error messages
3. Try force reload:
   load 'C:/path/to/GG_ExtraNesting/gg_extra_nesting.rb'


ERROR: Can't see N2 Playground
---------------------------------------------------------------
Solution: Use navigation tools
1. Menu: Dev: Focus N2
2. Or Ruby: GG_Cabinet::ExtraNesting::DevTools.focus_n2
3. Look for group named: __N2_Playground_ExtraNesting


ERROR: Playground offset is wrong
---------------------------------------------------------------
Solution: Check offset
1. Run: GG_Cabinet::ExtraNesting::DevTools.compare_roots
2. Offset should be 20000mm on X-axis
3. If wrong, reset: PlaygroundCreator.reset_playground(model)
